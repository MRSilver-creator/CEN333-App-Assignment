// ABOUT

What is QuickDispatch?

A cloud-connected dispatch tool built for small logistics teams. Dispatchers create delivery requests by tapping on a Google Map — the app computes distances, syncs to Firebase in real time across every device, and Gemini AI generates a professional 2-sentence courier briefing for each drop-off.

// FEATURES

What it does

📍

Map-Click Location

Tap anywhere on the Google Map to pin a delivery. Lat/lng captured instantly.

02
